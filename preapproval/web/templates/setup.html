One-time setup: add your Anthropic API key

The review assistant uses Claude to read forms and check websites. Before you can start a review, the {{ key_env }} environment variable must be set.

  1. Get a key from the Anthropic Console.
  2. In your terminal, set it and restart the app: 
    export {{ key_env }}="sk-ant-..."
python -m preapproval serve
  3. Reload this page and start your review.

The key is read from the environment only — it is never written to disk, logs, or reports.
